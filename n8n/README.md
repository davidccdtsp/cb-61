#N8N

## Arranque de la herramienta

```bash
docker compose up -d
```

Acceso en [http://localhost:5678](http://localhost:5678)

## Importar el flujo

### Desde fichero json
En la se seccion Overview -> Create workflow, en el menu desplegable en los tres puntos seleccionar import from file e importar el flujo contenido en el directorio flow.

### Desde url
Entrando en el siguiente enlace [https://n8n.io/workflows/6270-build-your-first-ai-agent/](https://n8n.io/workflows/6270-build-your-first-ai-agent/), haciendo click en el boton **Use for free** y seleccionando la opción **Import template to <Instancia-N8N> self-hosted instance.

Video ejemplo de importación de un flujo: [https://youtu.be/d2WzxdLm2-4](https://youtu.be/d2WzxdLm2-4)

## Configuracion del API Key
